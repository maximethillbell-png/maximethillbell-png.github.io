MaxPlay v4.2

application · desktop

Un lecteur vidéo local, propre et rapide, construit avec Electron et le moteur mpv. Pensé pour lire tes fichiers vidéo du disque avec une interface soignée.

À propos

C'est une vraie application de bureau (Electron + mpv), pas une page web — elle a besoin d'être installée pour tourner. Le code source complet, l'installeur et les builds sont dans le dossier MaxView-Electron/ sur ta machine.

  • Lecture de fichiers vidéo locaux via le moteur mpv.
  • Interface personnalisée (dossier UI-UX/).
  • Builds Windows et Linux disponibles dans dist/.

Comme c'est une app desktop volumineuse, elle n'est pas exécutable directement depuis le Cabinet — cette page sert de fiche.
