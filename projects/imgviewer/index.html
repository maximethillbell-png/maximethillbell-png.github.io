ImgViewer

outil · desktop

Une petite visionneuse d'image flottante, sans bordure, toujours au-dessus des autres fenêtres. Parfaite pour garder une référence à l'écran (une capture, un mockup) pendant qu'on travaille ailleurs.

Ce que ça fait

  • Fenêtre sans bordure, épinglée au-dessus de tout.
  • Colle une image directement depuis le presse-papier (Ctrl+V).
  • Se déplace, se redimensionne, opacité réglable, verrouillage.

Lancer

C'est une app Python (Tkinter + Pillow). Depuis le dossier du projet :

pip install pillow
python imgviewer.py

Une version .exe existe aussi dans le dossier dist/ du projet sur ta machine.
