Comment jouer ?

Devine le mob de Dofus du jour. Il change toutes les 24 heures. Une fois trouvé, reviens demain pour le suivant !

Mode classique

Tape le nom d'un mob pour révéler ses propriétés. La couleur des tuiles indique à quel point ta proposition est proche du mob mystère :

Propriétés

Type/race : la famille du mob (Bouftous, Blops, Kwaks…).
Boss ? : Mob de base, Boss, ou Avis de recherche.
Zone : la région où on le rencontre (Amakna, Cania, Frigost…).
Niveau : tranche de niveau ; ▲ / ▼ indiquent si le mob mystère est plus haut / plus bas.
Faiblesse : éléments contre lesquels il est vulnérable.
Résistance : éléments contre lesquels il résiste.
Invuln. : immunité totale à tous les éléments (Oui / Non).
