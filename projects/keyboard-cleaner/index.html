Keyboard Cleaner

outil · desktop

Désactive complètement le clavier pour pouvoir le nettoyer sans déclencher n'importe quoi. Un clic sur la fenêtre le réactive.

Lancer

App Python (Tkinter + pynput). La dépendance s'installe toute seule au premier lancement.

python keyboard_cleaner.py

Astuce : garde la souris à portée — c'est elle qui réactive le clavier.
