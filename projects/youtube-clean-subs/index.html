YouTube Clean Subscriptions

extension · navigateur

Une extension Chrome qui nettoie la page « Abonnements » de YouTube : elle supprime les sections « Pertinentes » et les Shorts pour ne garder que le flux chronologique de tes abonnements.

Installer

  • Télécharge les deux fichiers ci-dessous dans un même dossier.
  • Va sur chrome://extensions, active le mode développeur.
  • Clique « Charger l'extension non empaquetée » et sélectionne le dossier.
